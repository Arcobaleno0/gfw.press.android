
欢迎使用 GFW.Press 软件安卓版


请访问 http://gfw.press/GFW.Press.apk 下载安装包并安装

请访问 http://gfw.press 获取连接配置信息

配置填写完毕，点击打开运行开关


GFW.Press 软件安卓版使用HTTP全局代理模式，部分软件不走HTTP全局代理，如Twitter客户端等，需要进入这些软件内部设置代理，代理主机：127.0.0.1 端口：3128 或其它配置的端口
